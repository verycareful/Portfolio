# Sricharan Suresh — Portfolio

Personal portfolio of **Sricharan Suresh**, an Aspiring Quantum Engineer studying Computer Science + Quantum Computing (Honors) at SRM IST Vadapalani.

Building at the edge — from Ising Hamiltonians to NFC-based campus systems.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | [Next.js 16](https://nextjs.org/) (App Router, TypeScript) |
| Styling | [Tailwind CSS v4](https://tailwindcss.com/) |
| Animation | [Framer Motion](https://www.framer.com/motion/) |
| Background | HTML5 Canvas (quantum ambient particles) |
| Fonts | Syne · DM Sans · JetBrains Mono |
| Icons | [Lucide React](https://lucide.dev/) |
| Deployment | [Vercel](https://vercel.com/) |

---

## Sections

- **Hero** — Typewriter role cycle, quantum ambient background
- **About** — Stats, education, organisations
- **Research** — Published and in-review papers
- **Projects** — Filterable project grid (12 projects)
- **Lab Notebook** — Devlog entries + live "Currently" sidebar
- **Certifications** — Certs, badges, and a patent
- **Contact** — Links and mailto form

---

## Running Locally

```bash
git clone https://github.com/verycareful/Portfolio.git
cd Portfolio/portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
